Simulator::cCelestialBody class
#include <Spore/Simulator/cCelestialBody.h>

Represents celestial bodies in a solar system (except planets), that is, the star and the comets.

Contents

This is the actual graphical representation that you see in the solar system view, with the graphics itself being mpEffect. Celestial bodies are used by the cSolarSystem class.
