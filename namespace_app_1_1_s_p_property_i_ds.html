App::SPPropertyIDs namespace
#include <Spore/App/SPPropertyIDs.h>

This namespace only contains property IDs to be used in the AppProperties module.

Contents

  • Reference
    • Enums
    • Variables

These properties define basic configurations that should be fast and accessible in all the app. They were taken from the AppProperties.txt file.
